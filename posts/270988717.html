懒加载你的评论插件

YFun 2021年1月22日 共 709 字 预计 7 分钟
  1. 1. 前言
  2. 2. 实现
    1. 2.1. 修改评论区代码
    2. 2.2. JS 实现懒加载

前言

打开博客的文章页面，加载完毕就花费了一秒多的时长，打开控制台一看：

全是评论的表情包

再加上之前使用的是 Valine + LeanCloud 国际版，更是拖慢速度。

我就决定懒加载评论插件，在评论区进入（或即将进入）可视范围时再加载。

实现

提示
目前仅测试了 Twikoo 评论系统，其它评论系统理论上应该同样适用。

修改评论区代码

找到并修改你的评论区代码，将原先的评论区代码放入 raw 属性值中，此处仅提供示例参考：

<script src="https://cdn.jsdelivr.net/npm/twikoo@1.0.0/dist/twikoo.all.min.js"></script>
<div id="lazyload-comment" raw='
<!-- 将原先的评论区代码放入 `raw` 属性值中 -->
<!-- 注意：引入的评论插件代码文件不要懒加载，避免报错 -->
<div id="tcomment"></div>
<script>
    twikoo.init({
        envId: "your-env-id",
        el: "#tcomment",
        region: "ap-guangzhou",
        path: "window.location.pathname"
    });
</script>
'></div>
注意
引入的代码文件不要懒加载，避免报错。

JS 实现懒加载

提示
我这里给出的代码依赖 jQuery，因为我使用了它的选择器和一些 API，一定程度上会方便很多。
实际上原生 JS 也是可以达到同样的效果。

使用 jQuery 选择页面中的懒加载元素：

var content = $("#lazyload-comment");

再获取页面高度等数据，循环寻找出页面中所有的懒加载元素。

如果元素所在高度 小于等于 可见高度，就代表元素是可见的。

如果元素可见，就检查当前的内容是否被标记为已加载，避免重复的加载耗费资源。

如果未加载，就从 raw 属性中获取值，将其添加至 HTML 内容。

function load() {
    // 获取页面高度等数据
    var scrollTop = document.body.scrollTop || document.documentElement.scrollTop;
    var winTop = window.innerHeight;
    // For 循环寻找出页面中所有的懒加载元素
    for (var i = 0; i < content.length; i++) {
        // 如果元素所在高度 小于等于 窗口高度+滚动高度 (可见高度)
        if (content[i].offsetTop <= scrollTop + winTop) {
            // 如果当前的内容没有被标记为已加载
            if (!content.attr("load")) {
                // 从 RAW 属性中获取值
                var pageContentRaw = content.attr("raw");
                // 将其添加至 HTML 内容
                $(content[i]).html(pageContentRaw);
                content.attr("load",true);
            }
        }
    }
}

在页面刚加载完毕的时候先执行一次。再当页面滚动的时候重复执行。

// 页面加载时先执行一次
load();
// 当页面滚动的时候执行
window.onscroll = function () {
    load();
}

别忘了在页面中引入这个 JS！
